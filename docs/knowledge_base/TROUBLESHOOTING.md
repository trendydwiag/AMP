# Troubleshooting — Mengatasi Masalah Umum

## Tujuan

Dokumen ini membantu mengatasi masalah yang sering terjadi di AMP Studio.

## Kapan Digunakan

- Saat mengalami masalah
- Saat fitur tidak berfungsi
- Saat ada error

---

## 1. Radio Tidak Bunyi

### Gejala
- Audio tidak terdengar meski tombol Play sudah ditekan
- Status sudah LIVE tapi tidak ada suara

### Penyebab
- Koneksi internet terputus
- Stream URL salah
- RadioBoss tidak berjalan
- Volume komputer 0

### Solusi
1. **Periksa volume komputer** — Pastikan volume tidak 0
2. **Periksa koneksi internet** — Buka website lain untuk memastikan
3. **Periksa status stream** — Buka Radio > Dashboard
4. **Periksa RadioBoss** — Pastikan RadioBoss berjalan
5. **Periksa stream URL** — Buka Radio > Station > Edit
6. **Restart RadioBoss** — Matikan lalu nyalakan kembali
7. **Hubungi operator** — Jika masih tidak berfungsi

## 2. Login Gagal

### Gejala
- Pesan "Email atau kata sandi salah"
- Tidak bisa masuk ke AMP Studio

### Penyebab
- Email salah
- Kata sandi salah
- Akun terkunci
- Akun tidak aktif

### Solusi
1. **Periksa email** — Pastikan email benar
2. **Periksa Caps Lock** — Pastikan tidak aktif
3. **Klik "Lupa Kata Sandi"** — Reset kata sandi
4. **Hubungi admin** — Jika akun terkunci
5. **Periksa email spam** — Untuk link reset

## 3. Lupa Password

### Gejala
- Tidak ingat kata sandi
- Ingin mengubah kata sandi

### Solusi
1. Buka halaman login
2. Klik **Lupa Kata Sandi?**
3. Ketik email Anda
4. Klik **Kirim Link Reset**
5. Buka email Anda
6. Klik link di email
7. Ketik kata sandi baru
8. Klik **Simpan**

## 4. Artikel Tidak Muncul

### Gejala
- Artikel yang sudah ditulis tidak terlihat
- Artikel tidak muncul di website

### Penyebab
- Artikel masih berstatus Draf
- Artikel menunggu review
- Artikel ditolak
- Masalah caching

### Solusi
1. **Periksa status artikel** — Pastikan sudah Diterbitkan
2. **Periksa status review** — Pastikan sudah disetujui
3. **Muat ulang halaman** — Tekan F5
4. **Periksa kategori** — Pastikan kategori benar
5. **Periksa filter** — Pastikan tidak ada filter aktif

## 5. Podcast Tidak Muncul

### Gejala
- Episode podcast tidak terlihat
- Podcast tidak bisa diputar

### Penyebab
- Episode masih berstatus Draf
- File audio rusak
- Masalah koneksi internet

### Solusi
1. **Periksa status episode** — Pastikan sudah Diterbitkan
2. **Periksa file audio** — Pastikan file tidak rusak
3. **Muat ulang halaman** — Tekan F5
4. **Coba browser lain** — Untuk memastikan bukan masalah browser
5. **Periksa koneksi internet** — Pastikan stabil

## 6. Upload Gagal

### Gejala
- File gagal diunggah
- Pesan error saat upload

### Penyebab
- File terlalu besar
- Format file tidak didukung
- Koneksi internet terputus
- Server penuh

### Solusi
1. **Periksa ukuran file** — Maksimal 5 MB untuk gambar, 50 MB untuk audio
2. **Periksa format** — JPG, PNG, GIF untuk gambar; MP3, WAV untuk audio
3. **Periksa koneksi internet** — Pastikan stabil
4. **Coba file lain** — Untuk memastikan bukan masalah file
5. **Hubungi admin** — Jika masih gagal

## 7. Stream Offline

### Gejala
- Status stream: OFFLINE
- Radio tidak bisa didengarkan

### Penyebab
- RadioBoss berhenti
- Koneksi internet terputus
- Server streaming down
- Stream URL salah

### Solusi
1. **Periksa RadioBoss** — Pastikan berjalan
2. **Periksa koneksi internet** — Pastikan stabil
3. **Restart RadioBoss** — Matikan lalu nyalakan
4. **Periksa stream URL** — Pastikan benar
5. **Hubungi penyedia hosting** — Jika server down
6. **Gunakan backup stream** — Jika tersedia

## 8. Logo Tidak Berubah

### Gejala
- Logo baru tidak tampil
- Logo lama masih muncul

### Penyebab
- Cache browser
- Logo belum tersimpan
- Format file salah
- Ukuran file terlalu besar

### Solusi
1. **Bersihkan cache browser** — Ctrl+Shift+Delete
2. **Muat ulang halaman** — Tekan Ctrl+F5
3. **Periksa logo** — Pastikan sudah tersimpan
4. **Periksa format** — PNG atau JPG
5. **Periksa ukuran** — Maksimal 2 MB
6. **Coba browser lain** — Untuk memastikan

## Tips Umum

1. **Muat ulang halaman** — Tekan F5 atau Ctrl+F5
2. **Bersihkan cache browser** — Ctrl+Shift+Delete
3. **Coba browser lain** — Chrome, Firefox, Edge
4. **Periksa koneksi internet** — Buka website lain
5. **Hubungi admin** — Jika masalah berlanjut

## Catatan

- Simpan pesan error untuk laporan ke admin
- Jangan mencoba memperbaiki sendiri jika tidak yakin
- Hubungi tim teknis untuk masalah serius

## FAQ

**Q: Bagaimana cara membersihkan cache browser?**
A: Tekan Ctrl+Shift+Delete > Pilih "Cache" > Hapus.

**Q: Browser apa yang terbaik untuk AMP Studio?**
A: Chrome atau Firefox versi terbaru.

**Q: Bagaimana cara melaporkan masalah?**
A: Hubungi admin atau tim teknis melalui email/chat.

**Q: Berapa lama masalah akan diperbaiki?**
A: Tergantung tingkat kesulitan. Masalah ringan: 1-2 jam. Masalah serius: 1-2 hari.

## Best Practice

1. **Muat ulang halaman sebelum melaporkan masalah**
2. **Periksa koneksi internet terlebih dahulu**
3. **Coba browser lain untuk memastikan**
4. **Simpan pesan error untuk laporan**
5. **Hubungi admin dengan informasi yang lengkap**
