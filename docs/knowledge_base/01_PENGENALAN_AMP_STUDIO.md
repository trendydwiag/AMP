# Pengenalan AMP Studio

## Tujuan

Dokumen ini menjelaskan apa itu AMP Studio, fitur-fitur yang tersedia, dan cara kerja sistem ini secara umum.

## Apa itu AMP Studio?

AMP Studio adalah sistem pengelolaan konten untuk radio. Dengan AMP Studio, Anda bisa:

- Mengelola siaran radio langsung
- Menulis dan menerbitkan artikel
- Mengelola podcast
- Mengunggah gambar dan file
- Melihat statistik pendengar
- Mengelola jadwal siaran
- Bekerja sama dengan tim

## Siapa yang Menggunakan AMP Studio?

| Peran | Keterangan |
|-------|------------|
| Super Admin | Mengelola seluruh sistem dan partner |
| Partner Admin | Mengelola satu stasiun radio |
| Station Manager | Mengelola operasional stasiun |
| Program Director | Mengatur program dan jadwal |
| PenyiAR | Menyiarkan secara langsung |
| Editor Konten | Menulis artikel |
| Marketing | Mengelola promosi |
| Tim Sponsor | Mengelola sponsor |
| Operator Radio | Mengatur perangkat teknis |
| Pendengar | Mendengarkan radio dan membaca artikel |
| Guest | Mengunjungi website |

## Fitur Utama

### 1. Siaran Radio
- Siaran langsung 24 jam
- Pengaturan program dan jadwal
- Status LIVE/OFFLINE real-time
- Penghitung pendengar

### 2. Konten Artikel
- Penulisan artikel mudah
- Pengaturan SEO otomatis
- Status: Draf, Menunggu Review, Diterbitkan
- Kategori dan tag

### 3. Podcast
- Unggah episode podcast
- Kelola program podcast
- Pemutar audio built-in

### 4. Media
- Unggah gambar, audio, video
- Folder pengelolaan file
- Pencarian file cepat

### 5. Statistik
- Jumlah pendengar real-time
- Artikel paling banyak dibaca
- Program paling populer

### 6. Jadwal
- Kalender siaran
- Penugasan host
- Deteksi konflik jadwal

## Tampilan AMP Studio

AMP Studio memiliki tiga bagian utama:

### Sidebar (Menu Sisi)
Bagian kiri yang berisi menu navigasi. Menu yang ditampilkan tergantung peran Anda.

### Header (Bagian Atas)
Bagian atas yang berisi:
- Tombol menu
- Status siaran (LIVE/OFFLINE)
- Pencarian (tekan Ctrl+K)
- Notifikasi
- Pengaturan tema (terang/gelap)
- Menu pengguna

### Konten Utama
Bagian tengah yang berisi halaman yang sedang Anda buka.

## Tema

AMP Studio memiliki tiga tema:

1. **Terang** — Tema default dengan latar belakang putih
2. **Gelap** — Tema dengan latar belakang gelap, nyaman untuk malam hari
3. **Kopi** — Tema dengan nuansa coklat hangat

Untuk mengubah tema, klik ikon bulan/matahari di bagian atas.

## Pintasan Keyboard

| Tombol | Fungsi |
|--------|--------|
| Ctrl+K | Buka pencarian cepat |
| Esc | Tutup jendela yang terbuka |
| Tab | Pindah ke elemen berikutnya |

## Kapan Digunakan

- Saat ingin mengelola siaran radio
- Saat ingin menulis artikel
- Saat ingin mengunggah podcast
- Saat ingin melihat statistik
- Saat ingin mengatur jadwal

## Tips

- Gunakan tema gelap saat bekerja malam hari
- Gunakan pintasan Ctrl+K untuk pencarian cepat
- Periksa status siaran secara berkala

## Catatan

- AMP Studio bisa diakses dari komputer dan ponsel
- Gunakan browser Chrome atau Firefox untuk hasil terbaik
- Selalu logout setelah selesai bekerja

## Troubleshooting

Jika halaman tidak tampil:
1. Periksa koneksi internet
2. Muat ulang halaman (tekan F5)
3. Hubungi admin jika masih bermasalah

## FAQ

**Q: Apakah AMP Studio bisa diakses dari ponsel?**
A: Ya, AMP Studio bisa diakses dari ponsel melalui browser.

**Q: Berapa banyak pengguna yang bisa login bersamaan?**
A: Tidak ada batasan. Semua pengguna bisa login bersamaan.

**Q: Apakah perlu menginstal software tambahan?**
A: Tidak. AMP Studio berjalan di browser.
