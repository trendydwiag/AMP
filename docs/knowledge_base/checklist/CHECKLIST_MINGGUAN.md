# Checklist Mingguan

## Tujuan

Dokumen ini berisi daftar periksa yang harus dilakukan setiap minggu.

## Kapan Digunakan

- Setiap awal minggu (Senin)
- Saat merencanakan aktivitas mingguan
- Saat mengevaluasi minggu sebelumnya

---

## Daftar Periksa

### Awal Minggu (Senin)

#### Perencanaan
- [ ] Rencanakan konten minggu ini
- [ ] Buat jadwal konten minggu ini
- [ ] Tinjau jadwal siaran minggu ini
- [ ] Komunikasikan rencana ke tim

#### Pengaturan
- [ ] Periksa pengaturan radio
- [ ] Periksa pengaturan email
- [ ] Periksa backup minggu lalu
- [ ] Periksa statistik minggu lalu

### Tengah Minggu (Rabu)

#### Pemantauan
- [ ] Periksa statistik minggu ini
- [ ] Periksa artikel yang sudah diterbitkan
- [ ] Periksa performa konten
- [ ] Kumpulkan masukan dari tim

#### Perbaikan
- [ ] Perbaiki masalah yang ditemukan
- [ ] Perbarui konten jika perlu
- [ ] Tambah konten baru jika perlu
- [ ] Optimasi SEO jika perlu

### Akhir Minggu (Jumat)

#### Evaluasi
- [ ] Tinjau aktivitas minggu ini
- [ ] Evaluasi performa konten
- [ ] Evaluasi performa siaran
- [ ] Catat pencapaian dan masalah

#### Persiapan
- [ ] Siapkan konten minggu depan
- [ ] Siapkan jadwal minggu depan
- [ ] Backup data minggu ini
- [ ] Dokumentasikan perubahan

---

## Tips

- Rencanakan konten jauh hari
- Evaluasi secara rutin
- Dokumentasikan semua perubahan
- Komunikasikan dengan tim secara rutin
- Backup data secara berkala

## Catatan

- Checklist ini untuk role Administrator dan Partner Admin
- Role lain mungkin memiliki checklist yang berbeda
- Sesuaikan dengan kebutuhan spesifik

## Troubleshooting

| Masalah | Solusi |
|---------|--------|
| Statistik tidak lengkap | Periksa periode waktu |
| Backup gagal | Periksa ruang penyimpanan |
| Konten tidak performa | Evaluasi dan perbaiki |

## FAQ

**Q: Apakah harus melakukan semua checklist setiap minggu?**
A: Ya, untuk memastikan kualitas konten dan siaran.

**Q: Bagaimana jika ada libur?**
A: Sesuaikan jadwal checklist sesuai hari kerja.

**Q: Berapa lama waktu yang dibutuhkan?**
A: Biasanya 1-2 jam per minggu.

## Best Practice

1. **Rencanakan konten jauh hari**
2. **Evaluasi secara rutin**
3. **Dokumentasikan semua perubahan**
4. **Komunikasikan dengan tim secara rutin**
5. **Backup data secara berkala**
