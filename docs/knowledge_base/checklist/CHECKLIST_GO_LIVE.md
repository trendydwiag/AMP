# Checklist Go Live

## Tujuan

Dokumen ini berisi daftar periksa sebelum Partner bisa mulai beroperasi (go live).

## Kapan Digunakan

- Saat pertama kali mengaktifkan Partner
- Saat semua pengaturan sudah selesai
- Sebelum mulai siaran pertama

---

## Daftar Periksa

### Pengaturan Dasar
- [ ] Partner dibuat dan aktif
- [ ] Logo Partner diunggah
- [ ] Warna tema diatur
- [ ] Nama Partner sudah benar

### Domain dan Keamanan
- [ ] Domain dikonfigurasi
- [ ] SSL certificate aktif (https)
- [ ] DNS terkonfigurasi dengan benar
- [ ] Website bisa diakses melalui domain

### Email
- [ ] SMTP dikonfigurasi
- [ ] Email uji berhasil terkirim
- [ ] Email notifikasi aktif
- [ ] Email reset password berfungsi

### Radio
- [ ] Station dibuat
- [ ] Stream URL diatur
- [ ] RadioBoss terhubung
- [ ] Status stream: PLAYING
- [ ] Audio bisa didengar

### Pengguna
- [ ] Administrator Partner ditambahkan
- [ ] Administrator bisa login
- [ ] Role pengguna ditetapkan
- [ ] PenyiAR ditambahkan
- [ ] Program Director ditambahkan
- [ ] Editor Konten ditambahkan
- [ ] Operator Radio ditambahkan

### Program dan Jadwal
- [ ] Program siaran dibuat
- [ ] Jadwal siaran diatur
- [ ] Host ditugaskan ke program
- [ ] Tidak ada konflik jadwal

### Konten
- [ ] Artikel pertama ditulis
- [ ] Artikel sudah diterbitkan
- [ ] Gambar dan media diunggah
- [ ] SEO diatur dengan benar

### Backup
- [ ] Backup otomatis aktif
- [ ] Backup manual dilakukan
- [ ] File backup tersimpan dengan aman

### Pengujian
- [ ] Login berhasil dari berbagai perangkat
- [ ] Siaran radio berjalan lancar
- [ ] Website bisa diakses dari berbagai browser
- [ ] Artikel bisa dibaca dengan benar
- [ ] Email notifikasi terkirim
- [ ] Semua fitur berfungsi

---

## Setelah Go Live

1. Pantau statistik selama 1 minggu pertama
2. Kumpulkan masukan dari pengguna
3. Perbaiki masalah yang ditemukan
4. Dokumentasikan semua pengaturan
5. Jadwalkan backup rutin

---

## Tips

- Jangan terburu-buru go live
- Uji semua fitur sebelum go live
- Minta bantuan Super Admin jika ada masalah
- Dokumentasikan semua pengaturan
- Siapkan rencana darurat

## Catatan

- Go Live membutuhkan persetujuan dari Super Admin
- Semua item checklist harus selesai sebelum go live
- Jika ada masalah, segera hubungi tim teknis

## Troubleshooting

| Masalah | Solusi |
|---------|--------|
| Domain tidak bisa diakses | Periksa pengaturan DNS |
| SSL tidak aktif | Hubungi penyedia SSL |
| Email tidak terkirim | Periksa pengaturan SMTP |
| Stream tidak berjalan | Periksa RadioBoss dan koneksi |

## FAQ

**Q: Berapa lama proses go live?**
A: Biasanya 1-2 minggu, tergantung kesiapan tim.

**Q: Apakah bisa go live tanpa domain sendiri?**
A: Ya, bisa menggunakan subdomain dari AMP Studio.

**Q: Bagaimana jika ada masalah setelah go live?**
A: Hubungi tim teknis untuk bantuan.

## Best Practice

1. **Jangan terburu-buru go live**
2. **Uji semua fitur sebelum go live**
3. **Dokumentasikan semua pengaturan**
4. **Siapkan rencana darurat**
5. **Komunikasikan dengan tim secara rutin**
