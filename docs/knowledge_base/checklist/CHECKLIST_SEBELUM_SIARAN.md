# Checklist Sebelum Siaran

## Tujuan

Dokumen ini berisi daftar periksa yang harus dilakukan sebelum memulai siaran.

## Kapan Digunakan

- 30 menit sebelum siaran dimulai
- Saat mempersiapkan diri
- Saat memeriksa perangkat

---

## Daftar Periksa

### Persiapan Diri (30 Menit Sebelum)

#### PenyiAR
- [ ] Datang ke studio
- [ ] Buka AMP Studio
- [ ] Lihat jadwal hari ini
- [ ] Baca rundown siaran
- [ ] Baca script pembukaan
- [ ] Siapkan playlist lagu

#### Operator Radio
- [ ] Periksa status stream
- [ ] Pastikan RadioBoss berjalan
- [ ] Periksa koneksi internet
- [ ] Periksa perangkat audio
- [ ] Siapkan backup stream

### Pemeriksaan Perangkat (15 Menit Sebelum)

#### Audio
- [ ] Microphone terhubung
- [ ] Microphone berfungsi
- [ ] Headphone terhubung
- [ ] Headphone berfungsi
- [ ] Volume diatur dengan benar

#### Komputer
- [ ] Komputer menyala
- [ ] Internet berfungsi
- [ ] RadioBoss terbuka
- [ ] AMP Studio terbuka
- [ ] Tidak ada gangguan

#### Stream
- [ ] Stream URL benar
- [ ] Koneksi stream stabil
- [ ] Backup stream siap
- [ ] Status: PLAYING

### Pemeriksaan Konten (10 Menit Sebelum)

#### Playlist
- [ ] Playlist lengkap
- [ ] Urutan lagu benar
- [ ] Jingle siap
- [ ] Jeda iklan siap

#### Rundown
- [ ] Rundown lengkap
- [ ] Waktu sesuai
- [ ] Segmen lengkap
- [ ] Script siap

#### Sponsor
- [ ] Daftar sponsor lengkap
- [ ] Pesan sponsor siap
- [ ] Jingle sponsor siap

### Pemeriksaan Akhir (5 Menit Sebelum)

#### Status
- [ ] Status siaran: Siap
- [ ] Semua perangkat berfungsi
- [ ] Tidak ada masalah teknis
- [ ] Tim siap

#### Komunikasi
- [ ] Hubungi operator jika ada masalah
- [ ] Konfirmasi dengan tim
- [ ] Siap untuk siaran

---

## Tips

- Datang minimal 30 menit sebelum siaran
- Periksa semua perangkat sebelum mulai
- Siapkan backup untuk semua perangkat
- Komunikasikan masalah segera
- Jangan terburu-buru

## Catatan

- Jika ada masalah, segera hubungi operator
- Jika tidak bisa hadir, hubungi Station Manager minimal 24 jam sebelumnya
- Semua aktivitas siaran dicatat oleh sistem

## Troubleshooting

| Masalah | Solusi |
|---------|--------|
| Microphone tidak bunyi | Periksa kabel dan pengaturan audio |
| Internet tidak stabil | Hubungi operator untuk bantuan |
| RadioBoss tidak berfungsi | Restart RadioBoss |
| Stream tidak berjalan | Periksa URL stream dan koneksi |

## FAQ

**Q: Berapa lama sebelum siaran harus datang?**
A: Minimal 30 menit sebelum waktu siaran.

**Q: Bagaimana jika perangkat rusak?**
A: Hubungi operator segera untuk bantuan.

**Q: Bagaimana jika tidak bisa hadir?**
A: Hubungi Station Manager minimal 24 jam sebelumnya.

**Q: Apa yang harus dilakukan jika ada masalah?**
A: Hubungi operator atau tim teknis segera.
