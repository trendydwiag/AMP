# Checklist Setelah Siaran

## Tujuan

Dokumen ini berisi daftar periksa yang harus dilakukan setelah mengakhiri siaran.

## Kapan Digunakan

- Setelah siaran berakhir
- Saat mengakhiri aktivitas
- Saat mencatat laporan

---

## Daftar Periksa

### Penutupan Siaran

#### PenyiAR
- [ ] Selesaikan acara sesuai rundown
- [ ] Putar jingle penutup
- [ ] Ucapkan salam penutup
- [ ] Tekan tombol Stop/End di RadioBoss
- [ ] Pastikan status: OFFLINE

#### Operator Radio
- [ ] Periksa status stream
- [ ] Pastikan stream berhenti
- [ ] Periksa log error
- [ ] Catat masalah teknis

### Pemeriksaan Setelah Siaran

#### Status
- [ ] Status siaran: OFFLINE
- [ ] Tidak ada audio yang tersisa
- [ ] Perangkat dalam kondisi aman
- [ ] Studio dalam kondisi bersih

#### Data
- [ ] Catat durasi siaran
- [ ] Catat jumlah pendengar
- [ ] Catat masalah yang ditemukan
- [ ] Catat umpan balik pendengar

### Pembersihan

#### Studio
- [ ] Matikan perangkat audio
- [ ] Lepaskan kabel yang tidak perlu
- [ ] Rapikan studio
- [ ] Matikan komputer jika tidak digunakan

#### Dokumentasi
- [ ] Isi laporan siaran
- [ ] Catat masalah teknis
- [ ] Catat masukan dari pendengar
- [ ] Kirim laporan ke Station Manager

### Persiapan Besok

#### Persiapan
- [ ] Periksa jadwal besok
- [ ] Siapkan konten untuk besok
- [ ] Hubungi tim jika ada masalah
- [ ] Backup data jika perlu

---

## Tips

- Selalu isi laporan setelah siaran
- Catat semua masalah yang ditemukan
- Bersihkan studio setelah siaran
- Persiapkan diri untuk siaran besok
- Beri tahu tim tentang masalah yang ditemukan

## Catatan

- Laporan siaran penting untuk evaluasi
- Masalah teknis harus dilaporkan segera
- Studio harus bersih setelah siaran

## Troubleshooting

| Masalah | Solusi |
|---------|--------|
| Status tidak OFFLINE | Muat ulang atau hubungi operator |
| Data siaran tidak tersimpan | Catat manual dan laporkan |
| Perangkat tidak bisa dimatikan | Hubungi operator |

## FAQ

**Q: Apakah harus isi laporan setiap siaran?**
A: Ya, untuk evaluasi dan perbaikan.

**Q: Bagaimana cara mengisi laporan?**
A: Buka AMP Studio > Isi form laporan siaran.

**Q: Berapa lama waktu yang dibutuhkan?**
A: Biasanya 10-15 menit setelah siaran.

**Q: Bagaimana jika ada masalah teknis?**
A: Catat di laporan dan hubungi operator.

## Best Practice
