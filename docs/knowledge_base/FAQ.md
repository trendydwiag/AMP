## Umum

**Q1: Apa itu AMP Studio?**
A: AMP Studio adalah sistem manajemen konten untuk radio. Dengan AMP Studio, Anda bisa mengelola siaran, artikel, podcast, dan semua aktivitas digital.

**Q2: Siapa yang menggunakan AMP Studio?**
A: Super Admin, Partner Admin, Station Manager, Program Director, PenyiAR, Editor Konten, Marketing, Tim Sponsor, Operator Radio, dan Pendengar.

**Q3: Apakah AMP Studio bisa diakses dari ponsel?**
A: Ya, AMP Studio bisa diakses dari ponsel melalui browser.

**Q4: Browser apa yang terbaik untuk AMP Studio?**
A: Chrome atau Firefox versi terbaru.

**Q5: Apakah perlu menginstal software tambahan?**
A: Tidak. AMP Studio berjalan di browser.

**Q6: Berapa banyak pengguna yang bisa login bersamaan?**
A: Tidak ada batasan. Semua pengguna bisa login bersamaan.

**Q7: Apakah AMP Studio gratis?**
A: AMP Studio adalah produk berlangganan. Hubungi sales untuk informasi harga.

**Q8: Bagaimana cara menghubungi support?**
A: Hubungi melalui email support@ampstudio.id atau chat di website.

**Q9: Apakah ada panduan video?**
A: Ya, tersedia di YouTube Channel AMP Studio.

**Q10: Bagaimana cara memberikan masukan?**
A: Kirim masukan melalui formulir di website atau email feedback@ampstudio.id.

## Login dan Keamanan

**Q11: Bagaimana cara login?**
A: Buka website AMP Studio > Ketik email dan kata sandi > Klik Masuk.

**Q12: Lupa password, bagaimana cara reset?**
A: Klik "Lupa Kata Sandi?" di halaman login > Ketik email > Cek email untuk link reset.

**Q13: Akun terkunci, bagaimana cara membuka?**
A: Hubungi admin untuk membuka kunci akun.

**Q14: Bagaimana cara mengubah kata sandi?**
A: Buka Pengaturan > Keamanan > Ubah Kata Sandi.

**Q15: Apakah bisa login dari dua perangkat?**
A: Ya, tapi sesi yang lama akan berakhir.

**Q16: Bagaimana cara logout?**
A: Klik nama Anda di pojok kanan atas > Klik Keluar.

**Q17: Apakah administrator bisa melihat kata sandi saya?**
A: Tidak. Administrator hanya bisa mereset kata sandi.

**Q18: Berapa lama sesi login berakhir?**
A: Sesi berakhir setelah 8 jam tidak aktif.

**Q19: Bagaimana cara mengetahui akun saya sudah aman?**
A: Gunakan kata sandi yang kuat dan jangan bagikan kepada orang lain.

**Q20: Apakah ada verifikasi dua langkah?**
A: Saat ini belum tersedia. Akan ditambahkan di versi mendatang.

**Q71: Apakah AMP Studio mendukung SSL?**
A: Ya, AMP Studio mendukung SSL (https).

**Q72: Bagaimana cara mengatur domain?**
A: Buka Pengaturan > Domain > Ketik domain > Simpan.

**Q73: Apakah AMP Studio mendukung SMTP?**
A: Ya, AMP Studio mendukung SMTP untuk email.

**Q74: Bagaimana cara mengatur SMTP?**

**Q75: Apakah AMP Studio mendukung backup?**
A: Ya, AMP Studio mendukung backup manual dan otomatis.

**Q76: Bagaimana cara melakukan backup?**

**Q77: Bagaimana cara restore backup?**
A: Buka Partner > Edit > Restore > Pilih file backup.

**Q78: Apakah AMP Studio mendukung multi-bahasa?**
A: Saat ini hanya mendukung Bahasa Indonesia.

**Q79: Bagaimana cara mengatur timezone?**
A: Buka Pengaturan > Situs > Timezone.

**Q80: Apakah AMP Studio mobile-friendly?**

## RadioBoss

**Q81: Apa itu RadioBoss?**
A: RadioBoss adalah program komputer yang mengontrol siaran radio otomatis.

**Q82: Bagaimana cara menghubungkan RadioBoss ke AMP Studio?**
A: Buka RadioBoss > Pengaturan > API > Isi API URL dan Key dari AMP Studio.

**Q83: Apakah RadioBoss gratis?**
A: RadioBoss adalah program berbayar. Hubungi penyedia untuk informasi.

**Q84: Bagaimana cara mengganti stream URL?**
A: Buka AMP Studio > Radio > Station > Edit > Ubah Stream URL.

**Q85: Apakah ada backup stream?**
A: Ya, siapkan Backup Stream URL di pengaturan Station.

**Q86: Bagaimana cara restart stream?**
A: Buka AMP Studio > Radio > Dashboard > Restart Stream.

**Q87: Bagaimana cara melihat status stream?**
A: Buka AMP Studio > Radio > Dashboard > Lihat status.

**Q88: Apakah stream bisa otomatis restart?**
A: Ya, aktifkan Auto Restart di pengaturan Station.

**Q89: Bagaimana jika stream sering putus?**
A: Periksa kuota bandwidth dan koneksi internet server.

**Q90: Bagaimana cara melihat health check?**
A: Buka AMP Studio > Radio > Dashboard > Health Check.
