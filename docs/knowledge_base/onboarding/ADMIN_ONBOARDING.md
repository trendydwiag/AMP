# Admin Onboarding

## Tujuan

Dokumen ini membantu Administrator baru memulai menggunakan AMP Studio.

## Kapan Digunakan

- Saat pertama kali menjadi Administrator
- Saat ingin memahami semua fitur

---

## Langkah-langkah Onboarding

### Hari 1: Orientasi

#### Login Pertama
1. Buka AMP Studio
2. Login dengan akun yang diberikan
3. Ganti kata sandi default
4. Kenali tampilan dashboard

#### Kenali Sidebar
- Beranda: Dashboard utama
- Siaran: Program, Episode, Jadwal, Host
- Konten: Artikel, Kategori, Tag, Penulis, SEO
- Podcast: Program, Episode
- Media: Semua File, Upload, Folder, Tag
- Analytics: Statistik
- Pengaturan: Situs, Tampilan, Email, Bahasa, Keamanan, Pengguna

[Gambar Dashboard Admin]

### Hari 2: Pengelolaan Pengguna

#### Melihat Daftar Pengguna
1. Buka **Pengaturan > Pengguna**
2. Lihat daftar pengguna yang ada

#### Menambah Pengguna
1. Klik **Tambah Pengguna**
2. Isi informasi:
   - Nama lengkap
   - Email
   - Kata sandi sementara
   - Role (peran)
3. Simpan

#### Mengubah Pengguna
1. Pilih pengguna dari daftar
2. Klik **Edit**
3. Ubah informasi yang diperlukan
4. Simpan

[Gambar Menu Pengguna]

### Hari 3: Pengelolaan Konten

#### Melihat Artikel
1. Buka **Konten > Artikel**
2. Lihat daftar artikel

#### Menulis Artikel Baru
1. Klik **Artikel Baru**
2. Isi judul dan isi
3. Atur kategori dan tag
4. Unggah gambar sampul
5. Simpan sebagai draf

[Gambar Menu Konten]

### Hari 4: Pengelolaan Media

#### Melihat Semua File
1. Buka **Media > Semua File**
2. Lihat daftar file

#### Mengunggah File
1. Klik **Media > Upload**
2. Pilih file dari komputer
3. Tunggu hingga selesai

[Gambar Media Explorer]

### Hari 5: Pengaturan Sistem

#### Pengaturan Situs
1. Buka **Pengaturan > Situs**
2. Ubah informasi situs
3. Simpan

#### Pengaturan Tampilan
1. Buka **Pengaturan > Tampilan**
2. Ubah logo dan warna
3. Simpan

#### Pengaturan Email
1. Buka **Pengaturan > Email**
2. Konfigurasi SMTP
3. Kirim email uji

[Gambar Pengaturan Sistem]

## Fitur yang Harus Dikuasai

### Pengelolaan Pengguna
- Menambah pengguna
- Mengubah informasi pengguna
- Menonaktifkan pengguna
- Mengatur role pengguna

### Pengelolaan Konten
- Menulis artikel
- Mengedit artikel
- Mengunggah gambar
- Mengatur SEO
- Menerbitkan artikel

### Pengelolaan Media
- Mengunggah file
- Membuat folder
- Menghapus file
- Mencari file

### Pengaturan Sistem
- Mengubah logo
- Mengubah warna
- Mengatur email
- Mengatur domain

---

## Tips

- Kenali semua menu di sidebar
- Coba setiap fitur secara bertahap
- Jangan takut untuk bereksperimen
- Minta bantuan jika ada masalah
- Dokumentasikan pengaturan penting

## Catatan

- Administrator memiliki akses penuh ke semua fitur
- Semua perubahan dicatat oleh sistem
- Gunakan hak akses dengan bijak

## Troubleshooting

| Masalah | Solusi |
|---------|--------|
| Tidak bisa login | Hubungi Super Admin untuk reset |
| Fitur tidak muncul | Periksa role pengguna |
| Artikel tidak tersimpan | Muat ulang halaman |
| Gambar gagal diunggah | Periksa ukuran file |

## FAQ

**Q: Bagaimana cara menambah pengguna?**
A: Buka Pengaturan > Pengguna > Tambah Pengguna.

**Q: Bisa menghapus artikel?**
A: Ya, tapi lebih baik arsipkan daripada hapus.

**Q: Bagaimana cara mengubah logo?**
A: Buka Pengaturan > Tampilan > Ubah Logo.

**Q: Bagaimana cara mengatur email?**
A: Buka Pengaturan > Email > Konfigurasi SMTP.

## Best Practice

1. **Kenali semua fitur sebelum digunakan**
2. **Gunakan role yang sesuai untuk setiap pengguna**
3. **Backup data secara berkala**
4. **Periksa statistik setiap minggu**
5. **Dokumentasikan pengaturan penting**
