# Partner Onboarding

## Tujuan

Dokumen ini membantu Partner baru memulai menggunakan AMP Studio.

## Kapan Digunakan

- Saat pertama kali menjadi Partner
- Saat ingin mengatur Partner dari awal

---

## Langkah-langkah Onboarding

### Minggu 1: Pengaturan Dasar

#### Hari 1: Aktivasi Akun
1. Terima email undangan dari Super Admin
2. Klik link di email untuk membuat akun
3. Login ke AMP Studio
4. Ganti kata sandi default

#### Hari 2: Pengaturan Profil
1. Unggah logo Partner
2. Isi informasi perusahaan
3. Atur warna tema

#### Hari 3: Pengaturan Domain
1. Daftarkan domain (jika belum punya)
2. Hubungkan domain ke AMP Studio
3. Aktifkan SSL (https)

#### Hari 4: Pengaturan Email
1. Konfigurasi SMTP
2. Kirim email uji
3. Pastikan email notifikasi berfungsi

#### Hari 5: Pengaturan Radio
1. Buat Station
2. Hubungkan RadioBoss
3. Atur Stream URL
4. Test stream

### Minggu 2: Pembentukan Tim

#### Hari 1: Tambah Administrator
1. Tambah admin Partner
2. Beri akses yang sesuai
3. Pastikan admin bisa login

#### Hari 2: Tambah Program Director
1. Tambah Program Director
2. Beri akses ke program dan jadwal
3. Pastikan bisa mengakses fitur

#### Hari 3: Tambah PenyiAR
1. Tambah PenyiAR
2. Beri akses untuk melihat jadwal
3. Pastikan bisa melihat rundown

#### Hari 4: Tambah Editor
1. Tambah Editor Konten
2. Beri akses untuk menulis artikel
3. Pastikan bisa mengunggah gambar

#### Hari 5: Tambah Operator
1. Tambah Operator Radio
2. Beri akses ke pengaturan teknis
3. Pastikan bisa memantau stream

### Minggu 3: Konten Awal

#### Hari 1: Buat Program
1. Buat program pertama
2. Isi deskripsi program
3. Atur durasi

#### Hari 2: Buat Jadwal
1. Buat jadwal mingguan
2. Tugaskan host ke program
3. Periksa konflik jadwal

#### Hari 3: Tulis Artikel Pertama
1. Tulis artikel pengenalan
2. Unggah gambar
3. Atur SEO
4. Kirim untuk review

#### Hari 4: Unggah Podcast
1. Unggah episode pertama
2. Isi informasi podcast
3. Atur cover art

#### Hari 5: Siapkan Media
1. Unggah logo dan gambar
2. Buat folder untuk organisasi
3. Periksa semua file

### Minggu 4: Pengujian

#### Hari 1: Uji Login
1. Login semua pengguna
2. Pastikan semua bisa mengakses fitur yang sesuai
3. Perbaiki jika ada masalah

#### Hari 2: Uji Siaran
1. Mulai siaran uji coba
2. Pastikan audio berjalan
3. Periksa status di AMP Studio

#### Hari 3: Uji Website
1. Buka website Partner
2. Pastikan semua halaman bisa diakses
3. Periksa tampilan mobile

#### Hari 4: Uji Email
1. Kirim email notifikasi
2. Pastikan email masuk
3. Periksa spam folder

#### Hari 5: Go Live
1. Pastikan semua sudah selesai
2. Aktifkan Partner
3. Mulai siaran pertama

- [ ] Akun admin Partner dibuat
- [ ] Logo Partner diunggah
- [ ] Warna tema diatur
- [ ] Domain dikonfigurasi
- [ ] SSL aktif
- [ ] SMTP dikonfigurasi

### Tim
- [ ] Administrator Partner ditambahkan
- [ ] Program Director ditambahkan
- [ ] PenyiAR ditambahkan
- [ ] Editor Konten ditambahkan
- [ ] Operator Radio ditambahkan

### Konten
- [ ] Program dibuat
- [ ] Jadwal diatur
- [ ] Artikel pertama ditulis
- [ ] Podcast pertama diunggah
- [ ] Media disiapkan

### Pengujian
- [ ] Login berhasil
- [ ] Siaran radio berjalan
- [ ] Website bisa diakses
- [ ] Email notifikasi berfungsi
- [ ] Semua fitur berfungsi

---

## Tips

- Ikuti langkah-langkah secara berurutan
- Jangan terburu-buru
- Minta bantuan Super Admin jika ada masalah
- Dokumentasikan semua pengaturan
- Uji setiap perubahan

## Catatan

- Onboarding biasanya memakan waktu 2-4 minggu
- Setiap Partner memiliki kebutuhan berbeda
- Super Admin siap membantu jika ada masalah

## Troubleshooting

| Masalah | Solusi |
|---------|--------|
| Email undangan tidak masuk | Hubungi Super Admin |
| Logo gagal diunggah | Periksa ukuran file (maks 2 MB) |
| Stream tidak berjalan | Periksa RadioBoss dan koneksi |
| Website tidak bisa diakses | Periksa domain dan DNS |

## FAQ

**Q: Berapa lama proses onboarding?**
A: Biasanya 2-4 minggu, tergantung kesiapan tim.

**Q: Apakah harus menunggu semua selesai sebelum go live?**
A: Tidak. Bisa go live dengan pengaturan dasar terlebih dahulu.

**Q: Bagaimana jika ada masalah teknis?**
A: Hubungi Super Admin untuk bantuan.

**Q: Apakah bisa mengubah pengaturan setelah go live?**
A: Ya, pengaturan bisa diubah kapan saja.

## Best Practice

1. **Ikuti langkah-langkah secara berurutan**
2. **Dokumentasikan semua pengaturan**
3. **Uji setiap perubahan sebelum diterapkan**
4. **Komunikasikan dengan tim secara rutin**
5. **Minta bantuan jika ada masalah**
