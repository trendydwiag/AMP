# PenyiAR Onboarding

## Tujuan

Dokumen ini membantu PenyiAR baru memulai menggunakan AMP Studio.

## Kapan Digunakan

- Saat pertama kali menjadi PenyiAR
- Saat ingin memahami cara siaran

---

## Langkah-langkah Onboarding

### Hari 1: Orientasi

#### Login Pertama
1. Buka AMP Studio
2. Login dengan akun yang diberikan
3. Ganti kata sandi default
4. Kenali tampilan dashboard

#### Kenali Menu yang Tersedia
- Beranda: Dashboard dengan jadwal hari ini
- Siaran: Lihat jadwal dan program

[Gambar Dashboard PenyiAR]

### Hari 2: Memahami Jadwal

#### Melihat Jadwal
1. Buka AMP Studio
2. Lihat bagian **Jadwal Hari Ini** di dashboard
3. Klik **Lihat Semua** untuk jadwal lengkap

#### Memahami Jadwal
- **Waktu**: Jam siaran dimulai dan berakhir
- **Program**: Nama program yang ditayangkan
- **Host**: Nama penyiar yang bertugas
- **Status**: ON AIR / Akan Datang / Selesai

[Gambar Jadwal Hari Ini]

### Hari 3: Mempersiapkan Siaran

#### Persiapan Sebelum Siaran
1. Datang minimal 30 menit sebelum waktu siaran
2. Periksa jadwal hari ini
3. Siapkan playlist lagu
4. Baca script pembukaan
5. Hubungkan perangkat audio
6. Periksa koneksi internet

#### Membuka Studio
1. Login ke AMP Studio
2. Lihat persiapan siaran di dashboard
3. Siapkan perangkat

[Gambar Persiapan Siaran]

### Hari 4: Memulai Siaran

#### Memulai Siaran
1. Buka RadioBoss atau perangkat siaran
2. Pilih program yang sesuai jadwal
3. Tekan tombol **Play** atau **Start**
4. Status siaran akan berubah menjadi **LIVE**

#### Selama Siaran
- Ikuti rundown sesuai urutan
- Putar lagu sesuai playlist
- Baca script sesuai waktu
- Catat waktu setiap segmen

[Gambar Tombol Mulai Siaran]

### Hari 5: Mengakhiri Siaran

#### Mengakhiri Siaran
1. Selesaikan acara sesuai rundown
2. Putar penutup atau jingle
3. Tekan tombol **Stop** atau **End**
4. Status siaran akan berubah menjadi **OFFLINE**

#### Setelah Siaran
1. Catat laporan siaran (opsional)
2. Beri tahu operator jika ada masalah
3. Kembalikan perangkat ke tempat semula

[Gambar Tombol Akhiri Siaran]

## Tips

- Datang minimal 30 menit sebelum siaran
- Periksa semua perangkat sebelum mulai
- Ikuti rundown sesuai urutan
- Berikan energi positif saat siaran
- Catat masukan dari pendengar

## Catatan

- PenyiAR tidak bisa mengubah jadwal
- PenyiAR tidak bisa menghapus konten
- Semua aktivitas siaran dicatat oleh sistem

## Troubleshooting

| Masalah | Solusi |
|---------|--------|
| Microphone tidak bunyi | Periksa kabel dan pengaturan audio |
| Koneksi terputus | Hubungi operator untuk bantuan |
| Lagu tidak diputar | Periksa playlist di RadioBoss |
| Status tidak LIVE | Muat ulang halaman AMP Studio |

## FAQ

**Q: Bagaimana cara melihat jadwal saya?**
A: Login ke AMP Studio > Lihat jadwal di dashboard.

**Q: Apakah saya bisa mengubah jadwal?**
A: Tidak. Hubungi Station Manager untuk perubahan jadwal.

**Q: Bagaimana jika saya tidak bisa hadir?**
A: Hubungi Station Manager minimal 24 jam sebelumnya.

**Q: Di mana saya melihat script?**
A: Buka AMP Studio > Klik episode > Lihat bagian Script.
