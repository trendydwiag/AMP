# Panduan Pendengar

## Tujuan

Dokumen ini menjelaskan cara menggunakan website radio bagi pendengar.

## Kapan Digunakan

- Saat ingin mendengarkan radio online
- Saat ingin membaca artikel
- Saat ingin mendengarkan podcast
- Saat ingin melihat jadwal siaran

---

## 1. Cara Mendengarkan Radio

### Langkah 1: Buka Website

Buka browser ketik alamat website radio.

### Langkah 2: Klik Tombol Play

Di halaman utama, cari tombol **Play** atau **Dengarkan Sekarang**.

### Langkah 3: Dengarkan

Audio akan mulai diputar. Anda bisa:
- Mengatur volume
- Menjeda (pause)
- Melanjutkan (play)

[Gambar Pemutar Audio]

---

## 2. Cara Membuka Website

Ketik alamat website di browser:
- Chrome: Ketik alamat di bar URL
- Firefox: Ketik alamat di bar URL
- Safari: Ketik alamat di bar URL

**Tips**: Bookmark (simpan) website agar mudah diakses kembali.

---

## 3. Cara Memutar Audio

1. Buka website radio
2. Cari pemutar audio (biasanya di bagian atas atau bawah halaman)
3. Klik tombol **Play** (lingkaran dengan segitiga)
4. Audio akan mulai diputar
5. Klik **Pause** (dua garis vertikal) untuk menjeda

[Gambar Kontrol Audio]

---

## 4. Cara Melihat Jadwal

1. Buka website radio
2. Cari menu **Jadwal** atau **Schedule**
3. Lihat program yang akan datang

[Gambar Halaman Jadwal]

---

## 5. Cara Membaca Artikel

1. Buka website radio
2. Cari menu **Artikel** atau **Berita**
3. Klik judul artikel yang ingin dibaca
4. Baca isi artikel

[Gambar Daftar Artikel]

---

## 6. Cara Mendengarkan Podcast

1. Buka website radio
2. Cari menu **Podcast**
3. Pilih program podcast
4. Pilih episode yang ingin didengar
5. Klik tombol **Play**

[Gambar Halaman Podcast]

---

## 7. Cara Menghubungi Radio

1. Buka website radio
2. Cari menu **Kontak** atau **Hubungi Kami**
3. Isi formulir pesan
4. Kirim pesan

Atau hubungi melalui:
- Telepon: [nomor telepon]
- Email: [alamat email]
- WhatsApp: [nomor WhatsApp]
- Media Sosial: [akun media sosial]

[Gambar Halaman Kontak]

---

## Tips

- Gunakan koneksi internet yang stabil untuk kualitas audio terbaik
- Bookmark website agar mudah diakses
- Ikuti media sosial untuk info terbaru
- Berikan feedback melalui formulir kontak

## Catatan

- Mendengarkan radio membutuhkan koneksi internet
- Kualitas audio tergantung kecepatan internet
- Tidak perlu login untuk mendengarkan radio
- Tidak perlu login untuk membaca artikel

## Troubleshooting

| Masalah | Solusi |
|---------|--------|
| Audio tidak bunyi | Periksa volume komputer/ponsel |
| Audio terputus | Periksa koneksi internet |
| Website tidak tampil | Muat ulang halaman (F5) |
| Artikel tidak muncul | Coba buka artikel lain |
| Podcast tidak diputar | Periksa koneksi internet |

## FAQ

**Q: Apakah perlu login untuk mendengarkan radio?**
A: Tidak. Radio bisa didengarkan tanpa login.

**Q: Apakah bisa mendengarkan di ponsel?**
A: Ya, buka website di browser ponsel.

**Q: Bagaimana cara mengetahui jadwal siaran?**
A: Buka menu Jadwal di website.

**Q: Bisakah saya mendengarkan ulang siaran?**
A: Ya, melalui fitur Podcast.

**Q: Bagaimana cara menghubungi radio?**
A: Buka menu Kontak atau hubungi nomor yang tertera di website.

## Best Practice

1. **Gunakan koneksi internet yang stabil**
2. **Bookmark website untuk akses cepat**
3. **Ikuti media sosial untuk info terbaru**
4. **Dengarkan di waktu senggang untuk kualitas terbaik**
5. **Berikan feedback untuk perbaikan layanan**
