# Panduan Program Director

## Tujuan

Dokumen ini menjelaskan cara kerja Program Director secara detail.

## Kapan Digunakan

- Saat membuat program baru
- Saat mengatur episode
- Saat membuat playlist
- Saat menugaskan host
- Saat menyetujui konten

---

## 1. Mengatur Program

### Membuat Program Baru

1. Klik **Siaran** di sidebar
2. Klik **Program**
3. Klik **Tambah Program**
4. Isi informasi:
   - Nama program
   - Deskripsi
   - Durasi
   - Jenis (harian/mingguan)
5. Simpan

### Mengubah Program

1. Buka **Siaran > Program**
2. Pilih program
3. Klik **Edit**
4. Ubah informasi
5. Simpan

### Menghapus Program

1. Buka **Siaran > Program**
2. Pilih program
3. Klik **Hapus**
4. Konfirmasi

## 2. Mengatur Episode

### Membuat Episode Baru

1. Klik **Siaran > Episode**
2. Klik **Tambah Episode**
3. Isi informasi:
   - Judul episode
   - Nomor episode
   - Deskripsi
   - Program terkait
4. Simpan

### Mengubah Episode

1. Buka **Siaran > Episode**
2. Pilih episode
3. Klik **Edit**
4. Ubah informasi
5. Simpan

[Gambar Menu Episode]

---

## 3. Mengatur Playlist

### Membuat Playlist Baru

1. Klik **Siaran > Playlist**
2. Klik **Tambah Playlist**
3. Beri nama playlist
4. Tambah lagu/konten
5. Atur urutan
6. Simpan

### Mengatur Urutan Lagu

1. Buka playlist
2. Seret lagu ke posisi yang diinginkan
3. Simpan perubahan

[Gambar Menu Playlist]

---

## 4. Mengatur Host

### Menugaskan Host ke Program

1. Buka **Siaran > Program**
2. Pilih program
3. Klik **Edit**
4. Pilih **Host** dari daftar
5. Simpan

### Melihat Jadwal Host

1. Buka **Siaran > Host**
2. Lihat jadwal masing-masing host

[Gambar Menu Host]

## 5. Mengatur Schedule (Jadwal)

### Membuat Jadwal Baru

1. Klik **Siaran > Jadwal**
2. Klik **Tambah Jadwal**
3. Pilih program
4. Pilih hari dan waktu
5. Pilih host
6. Simpan

### Mengubah Jadwal

1. Buka **Siaran > Jadwal**
2. Pilih jadwal
3. Klik **Edit**
4. Ubah informasi
5. Simpan

### Menghapus Jadwal

1. Buka **Siaran > Jadwal**
2. Pilih jadwal
3. Klik **Hapus**
4. Konfirmasi

[Gambar Menu Jadwal]

---

## 6. Approval (Persetujuan)

### Melihat Konten yang Menunggu Review

1. Login ke AMP Studio
2. Di dashboard, lihat bagian **Menunggu Review**
3. Lihat daftar konten yang perlu disetujui

### Menyetujui Konten

1. Klik konten yang ingin disetujui
2. Baca dan tinjau konten
3. Klik **Setujui**
4. Konten akan diterbitkan

### Menolak Konten

1. Klik konten yang ingin ditolak
2. Baca konten
3. Klik **Tolak**
4. Berikan alasan penolakan
5. Kirim ke penulis

## Tips

- Rencanakan program jauh hari
- Buat playlist yang bervariasi
- Tinjau konten secara berkala
- Berikan umpan balik yang konstruktif

## Catatan

- Program Director bisa menyetujui konten
- Program Director bisa mengatur playlist
- Program Director tidak bisa mengatur pengaturan teknis

## Troubleshooting

| Masalah | Solusi |
|---------|--------|
| Program tidak ada jadwal | Buat jadwal baru |
| Konten perlu persetujuan | Tinjau dan setujui di dashboard |
| Playlist kosong | Tambahkan item ke playlist |
| Host tidak tersedia | Cari pengganti atau ubah jadwal |

## FAQ

**Q: Bagaimana cara membuat program baru?**
A: Buka Siaran > Program > Tambah Program.

**Q: Bagaimana cara menyetujui konten?**
A: Buka Dashboard > Menunggu Review > Pilih konten > Setujui.

**Q: Bisa mengubah playlist?**
A: Ya, buka Siaran > Playlist > Edit.

**Q: Bagaimana jika host tidak bisa hadir?**
A: Buka Siaran > Jadwal > Edit > Ganti host.

## Best Practice

1. **Rencanakan program minimal dua minggu sebelumnya**
2. **Buat template untuk jenis program yang berulang**
3. **Tinjau konten minimal sekali sehari**
4. **Dokumentasikan standar kualitas program**
5. **Komunikasikan ekspektasi ke tim konten**
