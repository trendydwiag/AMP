# Panduan Partner Admin

## Tujuan

Dokumen ini menjelaskan cara kerja Partner Admin secara detail.

## Kapan Digunakan

- Saat pertama kali menjadi admin
- Saat mengelola pengguna
- Saat mengelola konten
- Saat mengatur tampilan website

---

## 1. Cara Login

1. Buka browser, ketik alamat website AMP Studio
2. Ketik **email** dan **kata sandi**
3. Klik **Masuk**

Anda akan masuk ke Dashboard.

[Gambar Halaman Login]

---

## 2. Dashboard

Dashboard menampilkan ringkasan aktivitas:
- Jumlah pendengar saat ini
- Artikel hari ini
- Jadwal hari ini
- Notifikasi

[Gambar Dashboard Partner Admin]

## 3. Mengelola Pengguna

### Melihat Daftar Pengguna

1. Klik **Pengaturan** di sidebar
2. Klik **Pengguna**
3. Lihat daftar pengguna

### Menambah Pengguna Baru

1. Klik **Tambah Pengguna**
2. Isi informasi:
   - Nama lengkap
   - Email
   - Kata sandi sementara
   - Role (peran)
3. Klik **Simpan**

### Mengubah Informasi Pengguna

1. Pilih pengguna dari daftar
2. Klik **Edit**
3. Ubah informasi yang diperlukan
4. Simpan

### Menonaktifkan Pengguna

1. Pilih pengguna
2. Klik **Nonaktifkan**
3. Konfirmasi

[Gambar Menu Pengguna]

---

## 4. Mengelola Konten

### Artikel

1. Klik **Konten** di sidebar
2. Klik **Artikel**
3. Lihat daftar artikel

Untuk membuat artikel baru, lihat [Panduan Editor Konten](EDITOR_KONTEN_GUIDE.md).

### Kategori

1. Klik **Konten > Kategori**
2. Tambah/edit/hapus kategori

### Tag

1. Klik **Konten > Tag**
2. Tambah/edit/hapus tag

[Gambar Menu Konten]

---

## 5. Mengatur Menu Website

### Langkah 1: Buka Pengaturan

1. Klik **Pengaturan** di sidebar
2. Klik **Tampilan**

### Langkah 2: Atur Menu

1. Pilih menu yang ingin diubah
2. Ubah urutan dengan drag and drop
3. Tambah/hapus item menu
4. Simpan

[Gambar Pengaturan Menu]

---

## 6. Mengubah Logo

1. Klik **Pengaturan > Tampilan**
2. Klik **Ubah Logo**
3. Pilih gambar dari komputer
4. Pratinjau logo
5. Klik **Simpan**

[Gambar Upload Logo]

---

## 7. Mengubah Warna

1. Klik **Pengaturan > Tampilan**
2. Klik **Sesuaikan Warna**
3. Pilih warna primer dan sekunder
4. Pratinjau perubahan
5. Klik **Simpan**

[Gambar Pengaturan Warna]

---

## 8. Mengelola Media

### Melihat Semua File

1. Klik **Media** di sidebar
2. Klik **Semua File**
3. Lihat daftar file

### Mengunggah File

1. Klik **Media > Upload**
2. Pilih file dari komputer
3. Tunggu hingga selesai
4. File akan tersimpan

### Membuat Folder

1. Klik **Media > Folder**
2. Klik **Tambah Folder**
3. Beri nama folder
4. Simpan

[Gambar Media Explorer]

---

## 9. Melihat Statistik

1. Klik **Analytics** di sidebar
2. Lihat grafik dan angka:
   - Jumlah pendengar
   - Artikel paling banyak dibaca
   - Program paling populer
   - Waktu siaran paling ramai

[Gambar Dashboard Analytics]

## Tips

- Buat pengguna dengan role yang sesuai
- Periksa statistik setiap minggu
- Atur jadwal konten secara rutin
- Backup data secara berkala

## Catatan

- Partner Admin hanya bisa mengelola Partner-nya sendiri
- Tidak bisa mengakses Django Admin
- Semua perubahan dicatat oleh sistem

## Troubleshooting

| Masalah | Solusi |
|---------|--------|
| Pengguna tidak login | Periksa status akun |
| Artikel tidak muncul | Periksa status publikasi |
| Media gagal diunggah | Periksa ukuran file (maks 5 MB) |
| Logo tidak berubah | Muat ulang halaman |
| Statistik tidak update | Tunggu beberapa menit |

## FAQ

**Q: Bisa menambah pengguna sebanyak mungkin?**
A: Ya, tidak ada batasan jumlah pengguna.

**Q: Bagaimana cara menghapus pengguna?**
A: Buka Pengaturan > Pengguna > Pilih pengguna > Nonaktifkan.

**Q: Bisa mengubah warna website?**
A: Ya, buka Pengaturan > Tampilan > Sesuaikan Warna.

**Q: Bagaimana cara melihat statistik?**
A: Buka Analytics di sidebar.

**Q: Bisa menghapus artikel yang sudah diterbitkan?**
A: Ya, tapi lebih baik arsipkan daripada hapus.

## Best Practice

1. **Buat akun untuk setiap anggota tim**
2. **Gunakan role yang sesuai untuk setiap pengguna**
3. **Atur jadwal konten mingguan**
4. **Periksa statistik setiap minggu**
5. **Backup data secara berkala**
