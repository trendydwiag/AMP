# AMP Studio — Pengetahuan Dasar

## Tentang Dokumen Ini

Dokumen ini adalah panduan lengkap untuk menggunakan AMP Studio. AMP Studio adalah sistem manajemen konten yang digunakan untuk mengelola radio, podcast, artikel, dan semua aktivitas digital.

Dokumen ini ditulis untuk semua pengguna, baik yang sudah terbiasa menggunakan komputer maupun yang baru pertama kali.

---

## Cara Menggunakan Dokumen Ini

1. **Mulai dari bagian Pengenalan** jika Anda baru pertama kali menggunakan AMP Studio.
2. **Buka bagian yang sesuai dengan peran Anda** untuk panduan khusus.
3. **Gunakan Daftar Isi** untuk menemukan topik yang Anda butuhkan.
4. **Lihat FAQ** jika memiliki pertanyaan umum.
5. **Lihat Troubleshooting** jika mengalami masalah.
