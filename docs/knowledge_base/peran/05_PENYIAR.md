# PenyiAR

## Tujuan

Dokumen ini menjelaskan cara kerja PenyiAR di AMP Studio. PenyiAR menyiarkan secara langsung dan mengelola siaran harian.

## Peran PenyiAR

PenyiAR bertanggung jawab atas:
- Menyiarkan secara langsung
- Mengikuti jadwal siaran
- Mengelola playlist saat siaran
- Melihat rundown dan script
- Berinteraksi dengan pendengar

## Kapan Digunakan

## Cara Melihat Jadwal

1. Klik **Siaran** di sidebar
2. Klik **Jadwal**
3. Cari jadwal untuk hari ini

[Gambar Menu Jadwal]

## Cara Membuka Studio

1. Login ke AMP Studio
2. Lihat jadwal hari ini di dashboard
3. Siapkan playlist dan rundown
4. Hubungkan perangkat audio
5. Mulai siaran

[Gambar Dashboard PenyiAR]

## Cara Memulai Siaran

### Persiapan
1. Periksa jadwal hari ini
2. Siapkan playlist lagu
3. Siapkan script pembukaan
4. Hubungkan microphone
5. Periksa koneksi internet

### Memulai
1. Buka RadioBoss atau perangkat siaran
2. Pilih program yang sesuai jadwal
3. Tekan tombol **Play** atau **Start**
4. Status siaran akan berubah menjadi **LIVE**

[Gambar Tombol Mulai Siaran]

## Cara Melihat Playlist

## Cara Melihat Script

1. Buka AMP Studio

[Gambar Script Siaran]

## Cara Mengakhiri Siaran

1. Selesaikan acara sesuai rundown
2. Putar penutup atau jingle
3. Tekan tombol **Stop** atau **End** di RadioBoss
4. Status siaran akan berubah menjadi **OFFLINE**
5. Catat laporan siaran jika diperlukan

## Tips

- Persiapkan diri minimal 30 menit sebelum siaran
- Periksa semua perangkat sebelum mulai
- Ikuti rundown sesuai urutan
- Catat hal-hal penting selama siaran
- Beri tahu operator jika ada masalah teknis

## Catatan

- PenyiAR tidak bisa mengubah jadwal
- PenyiAR tidak bisa menghapus konten
- Semua aktivitas siaran dicatat oleh sistem

## Troubleshooting

| Masalah | Solusi |
|---------|--------|
| Microphone tidak bunyi | Periksa kabel dan pengaturan audio |
| Koneksi terputus | Hubungi operator untuk bantuan |
| Lagu tidak diputar | Periksa playlist di RadioBoss |
| Status tidak LIVE | Muat ulang halaman AMP Studio |

## FAQ

**Q: Bagaimana cara melihat jadwal saya?**
A: Buka AMP Studio > Lihat jadwal di dashboard atau menu Siaran > Jadwal.

**Q: Apakah saya bisa mengubah jadwal?**
A: Tidak. Hubungi Station Manager untuk perubahan jadwal.

**Q: Bagaimana jika saya tidak bisa hadir?**
A: Hubungi Station Manager minimal 24 jam sebelumnya.

**Q: Di mana saya melihat script?**
A: Buka AMP Studio > Klik episode > Lihat bagian Script.

## Best Practice

1. **Datang minimal 30 menit sebelum siaran**
2. **Periksa semua perangkat sebelum mulai**
3. **Ikuti rundown sesuai urutan**
4. **Berikan energi positif saat siaran**
5. **Catat masukan dari pendengar**
