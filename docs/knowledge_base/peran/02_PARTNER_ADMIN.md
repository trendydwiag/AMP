# Partner Admin

## Tujuan

Dokumen ini menjelaskan cara kerja Partner Admin di AMP Studio. Partner Admin mengelola satu stasiun radio (Partner).

## Peran Partner Admin

Partner Admin bertanggung jawab atas:
- Mengelola pengguna di Partner-nya
- Mengelola konten (artikel, podcast)
- Mengatur tampilan website
- Mengelola media
- Memantau statistik

## Kapan Digunakan

- Saat mengelola pengguna stasiun
- Saat mengatur tampilan website
- Saat memantau aktivitas konten
- Saat mengelola sponsor

## Tips

- Buat pengguna dengan role yang sesuai
- Atur jadwal konten secara rutin
- Periksa statistik setiap minggu
- Backup data secara berkala

## Catatan

- Partner Admin hanya bisa mengelola Partner-nya sendiri
- Tidak bisa mengakses Django Admin
- Tidak bisa menghapus Partner

## Troubleshooting

| Masalah | Solusi |
|---------|--------|
| Pengguna tidak bisa login | Periksa status akun dan role |
| Artikel tidak muncul | Periksa status publikasi |
| Media gagal diunggah | Periksa ukuran file |
| Statistik tidak update | Tunggu beberapa menit atau muat ulang |

## FAQ

**Q: Bagaimana cara menambah pengguna baru?**
A: Buka Pengaturan > Pengguna > Tambah Pengguna.

**Q: Bisa mengubah logo?**
A: Ya, buka Pengaturan > Tampilan > Logo.

**Q: Bagaimana cara melihat statistik?**
A: Buka Analytics di sidebar.

## Best Practice

1. **Buat akun untuk setiap anggota tim**
2. **Gunakan role yang sesuai untuk setiap pengguna**
3. **Atur jadwal konten mingguan**
4. **Periksa email notifikasi secara berkala**
5. **Dokumentasikan pengaturan penting**
