# Station Manager

## Tujuan

Dokumen ini menjelaskan cara kerja Station Manager di AMP Studio. Station Manager mengelola operasional stasiun radio.

## Peran Station Manager

Station Manager bertanggung jawab atas:
- Mengelola program siaran
- Mengatur jadwal siaran
- Memantau kualitas siaran
- Mengelola host/penyiar
- Melaporkan statistik

## Kapan Digunakan

- Saat mengatur program siaran
- Saat membuat jadwal mingguan
- Saat memantau kualitas siaran
- Saat mengelola host

## Tips

- Buat jadwal jauh hari sebelum siaran
- Komunikasikan perubahan jadwal ke tim
- Periksa status siaran sebelum jam siaran
- Dokumentasikan perubahan program

## Catatan

- Station Manager bisa mengubah jadwal siaran
- Station Manager bisa menugaskan host ke program
- Station Manager tidak bisa menghapus program

## Troubleshooting

| Masalah | Solusi |
|---------|--------|
| Jadwal konflik | Ubah waktu salah satu program |
| Host tidak tersedia | Hubungi host atau cari pengganti |
| Siaran terganggu | Hubungi Operator Radio |

## FAQ

**Q: Bagaimana cara membuat jadwal baru?**
A: Buka Siaran > Jadwal > Tambah Jadwal.

**Q: Bisa mengubah waktu siaran?**
A: Ya, buka Siaran > Jadwal > Pilih Jadwal > Edit.

**Q: Bagaimana cara menugaskan host?**
A: Buka Siaran > Jadwal > Pilih Jadwal > Pilih Host.

## Best Practice

1. **Buat jadwal minimal satu minggu sebelumnya**
2. **Konfirmasi ketersediaan host sebelum menugaskan**
3. **Periksa konflik jadwal sebelum menyimpan**
4. **Komunikasikan perubahan jadwal ke seluruh tim**
5. **Dokumentasikan setiap perubahan program**
