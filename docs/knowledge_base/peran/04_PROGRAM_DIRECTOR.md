# Program Director

## Tujuan

Dokumen ini menjelaskan cara kerja Program Director di AMP Studio. Program Director mengatur konten siaran dan program.

## Peran Program Director

Program Director bertanggung jawab atas:
- Merancang program siaran
- Mengatur konten episode
- Mengelola playlist
- Menugaskan host
- Menyetujui konten

## Yang Bisa Dilakukan Program Director

## Tips

- Rencanakan program jauh hari
- Buat playlist yang bervariasi
- Tinjau konten secara berkala
- Berikan umpan balik ke tim

## Catatan

- Program Director bisa menyetujui konten
- Program Director bisa mengatur playlist
- Program Director tidak bisa mengatur pengaturan teknis

## Troubleshooting

| Masalah | Solusi |
|---------|--------|
| Program tidak ada jadwal | Buat jadwal baru untuk program |
| Konten perlu persetujuan | Tinjau dan setujui di dashboard |
| Playlist kosong | Tambahkan item ke playlist |

## FAQ

**Q: Bagaimana cara membuat program baru?**
A: Buka Siaran > Program > Tambah Program.

**Q: Bagaimana cara menyetujui konten?**
A: Buka Dashboard > Menunggu Review > Pilih konten > Setujui.

**Q: Bisa mengubah playlist?**
A: Ya, buka Siaran > Playlist > Edit.

## Best Practice

1. **Rencanakan program minimal dua minggu sebelumnya**
2. **Buat template untuk jenis program yang berulang**
3. **Tinjau konten minimal sekali sehari**
4. **Dokumentasikan standar kualitas program**
5. **Komunikasikan ekspektasi ke tim konten**
