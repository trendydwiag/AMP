# Editor Konten

## Tujuan

Dokumen ini menjelaskan cara kerja Editor Konten di AMP Studio. Editor Konten menulis, mengedit, dan menerbitkan artikel.

## Peran Editor Konten

Editor Konten bertanggung jawab atas:
- Menulis artikel
- Mengedit artikel
- Mengunggah gambar
- Mengatur SEO
- Menerbitkan artikel

## Yang Bisa Dilakukan Editor Konten

| Fitur | Akses |
|-------|-------|
| Dashboard | ✓ Lihat |
| Kelola Artikel | ✓ Penuh |
| Kelola Kategori | ✓ Lihat |
| Kelola Tag | ✓ Penuh |
| Kelola Penulis | ✓ Lihat |
| Kelola SEO | ✓ Penuh |
| Kelola Media | ✓ Upload |

## Cara Menulis Artikel

### Langkah 1: Buka Form Artikel

1. Klik **Konten** di sidebar
2. Klik **Artikel**
3. Klik **Artikel Baru**

[Gambar Tombol Artikel Baru]

### Langkah 2: Isi Judul

Ketik judul artikel di kolom "Judul". Judul harus jelas dan menarik.

[Gambar Kolom Judul]

### Langkah 3: Tulis Isi Artikel

Gunakan editor untuk menulis isi artikel. Anda bisa:
- Menambah paragraf
- Menambah heading (judul kecil)
- Menambah gambar
- Menambah link

[Gambar Editor Artikel]

### Langkah 4: Atur Kategori

Pilih kategori yang sesuai untuk artikel Anda.

[Gambar Pilihan Kategori]

### Langkah 5: Tambah Tag

Ketik tag yang relevan. Tag membantu pembaca menemukan artikel.

[Gambar Kolom Tag]

### Langkah 6: Atur Gambar Sampul

1. Klik **Pilih Gambar**
2. Pilih gambar dari komputer atau media
3. Tunggu hingga gambar terunggah

[Gambar Upload Gambar Sampul]

### Langkah 7: Simpan sebagai Draf

Klik **Simpan** untuk menyimpan sebagai draf. Artikel belum bisa dilihat orang lain.

[Gambar Tombol Simpan]

## Cara Mengedit Artikel

1. Buka **Konten > Artikel**
2. Cari artikel yang ingin diedit
3. Klik **Edit** atau klik judul artikel
4. Ubah konten sesuai kebutuhan
5. Klik **Simpan**

[Gambar Tombol Edit]

## Cara Mengunggah Gambar

### Dalam Artikel
1. Buka editor artikel
2. Klik ikon gambar di toolbar
3. Pilih gambar dari komputer
4. Tunggu hingga terunggah

### Dari Media
1. Buka **Media > Upload**
2. Pilih gambar dari komputer
3. Tunggu hingga selesai

[Gambar Upload Gambar]

## Cara Mengatur SEO

### Langkah 1: Buka Pengaturan SEO

Di form artikel, gulir ke bawah ke bagian SEO.

### Langkah 2: Isi Meta Deskripsi

Ketik deskripsi singkat tentang artikel (maksimal 160 karakter).

### Langkah 3: Atur URL Slug

URL slug adalah alamat artikel di browser. Contoh: `tips-productivitas-kerja`

### Langkah 4: Pratinjau

Lihat pratinjau bagaimana artikel akan tampil di Google.

[Gambar Pengaturan SEO]

## Cara Menerbitkan Artikel

### Langkah 1: Periksa Artikel

Sebelum menerbitkan, pastikan:
- Judul sudah benar
- Isi sudah lengkap
- Gambar sudah terunggah
- SEO sudah diatur

### Langkah 2: Kirim untuk Review

Klik **Kirim untuk Review**. Artikel akan menunggu persetujuan.

[Gambar Tombol Kirim untuk Review]

### Langkah 3: Setelah Disetujui

Setelah disetujui oleh Program Director atau admin, artikel akan diterbitkan.

## Tips

- Tulis artikel dengan bahasa yang jelas dan mudah dipahami
- Gunakan gambar yang berkualitas dan relevan
- Atur SEO agar artikel mudah ditemukan di Google
- Periksa ejaan sebelum mengirim untuk review
- Simpan draf secara berkala

## Catatan

- Artikel yang sudah diterbitkan tidak bisa diubah tanpa persetujuan
- Gambar harus berukuran maksimal 5 MB
- Format gambar yang didukung: JPG, PNG, GIF, WebP

## Troubleshooting

| Masalah | Solusi |
|---------|--------|
| Gambar gagal diunggah | Periksa ukuran file (maks 5 MB) |
| Artikel tidak tersimpan | Muat ulang halaman dan coba lagi |
| SEO tidak muncul | Pastikan meta deskripsi sudah diisi |
| Editor tidak berfungsi | Muat ulang halaman |

## FAQ

**Q: Bisa menerbitkan artikel langsung?**
A: Tidak. Artikel harus disetujui terlebih dahulu oleh admin atau Program Director.

**Q: Berapa panjang artikel yang disarankan?**
A: Minimal 300 kata. Idealnya 800-1500 kata.

**Q: Bisa menghapus artikel yang sudah diterbitkan?**
A: Ya, tapi lebih baik arsipkan daripada hapus.

**Q: Bagaimana cara melihat statistik artikel?**
A: Buka artikel > Lihat bagian Statistik di pojok kanan atas.

## Best Practice

1. **Tulis dengan bahasa yang mudah dipahami**
2. **Gunakan gambar yang relevan dan berkualitas**
3. **Atur SEO untuk setiap artikel**
4. **Periksa ejaan sebelum mengirim untuk review**
5. **Simpan draf secara berkala**
