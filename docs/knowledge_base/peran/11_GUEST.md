# Guest

## Tujuan

Dokumen ini menjelaskan cara menggunakan website radio bagi pengunjung (guest) yang belum memiliki akun.

## Kapan Digunakan

- Saat pertama kali mengunjungi website
- Saat ingin melihat informasi umum
- Saat ingin mendengarkan radio
- Saat ingin membaca artikel

## Yang Bisa Dilakukan Guest

| Fitur | Akses |
|-------|-------|
| Mendengarkan Radio | ✓ Ya |
| Membaca Artikel | ✓ Ya |
| Melihat Jadwal | ✓ Ya |
| Melihat Halaman Tentang | ✓ Ya |
| Menghubungi Radio | ✓ Ya |
| Login | ✓ Ya (jika punya akun) |
| Mengelola Konten | ✗ Tidak |
| Mengelola Siaran | ✗ Tidak |
| Mengelola Pengguna | ✗ Tidak |

## Cara Mengunjungi Website

1. Buka browser
2. Ketik alamat website
3. Website akan terbuka

[Gambar Halaman Utama Website]

## Yang Tersedia untuk Guest

### Halaman Utama
- Informasi singkat tentang radio
- Tombol play radio
- Artikel terbaru
- Jadwal siaran

### Menu Navigasi
- **Beranda**: Kembali ke halaman utama
- **Artikel**: Daftar artikel berita
- **Podcast**: Daftar podcast
- **Jadwal**: Jadwal siaran
- **Tentang Kami**: Informasi tentang radio
- **Kontak**: Cara menghubungi radio

[Gambar Menu Navigasi Website]

## Cara Mendengarkan Radio

1. Buka website
2. Cari tombol **Play** atau **Dengarkan Sekarang**
3. Klik tombol tersebut
4. Audio akan mulai diputar

[Gambar Tombol Play]

## Cara Membaca Artikel

1. Buka website
2. Klik menu **Artikel** atau **Berita**
3. Pilih artikel yang ingin dibaca
4. Baca isi artikel

[Gambar Daftar Artikel]

## Cara Melihat Jadwal

1. Buka website
2. Klik menu **Jadwal**
3. Lihat program yang akan datang

[Gambar Halaman Jadwal]

## Cara Menghubungi Radio

1. Buka website
2. Klik menu **Kontak**
3. Isi formulir pesan
4. Kirim pesan

Atau hubungi melalui informasi kontak yang tersedia di website.

[Gambar Halaman Kontak]

## Membuat Akun (Opsional)

Jika Anda ingin memiliki akun untuk fitur lebih banyak:

1. Buka halaman login
2. Klink **Daftar** atau **Buat Akun**
3. Isi formulir pendaftaran
4. Verifikasi email (jika diperlukan)
5. Login dengan akun baru

[Gambar Halaman Daftar]

## Tips

- Bookmark website untuk akses cepat
- Ikuti media sosial untuk info terbaru
- Berikan feedback melalui formulir kontak
- Buat akun untuk fitur lebih banyak

## Catatan

- Guest tidak perlu login untuk mendengarkan radio
- Guest tidak perlu login untuk membaca artikel
- Guest tidak bisa mengelola konten atau siaran

## Troubleshooting

| Masalah | Solusi |
|---------|--------|
| Website tidak tampil | Periksa alamat website dan koneksi internet |
| Audio tidak bunyi | Periksa volume dan koneksi internet |
| Artikel tidak bisa dibuka | Coba buka artikel lain |
| Formulir kontak tidak terkirim | Periksa koneksi internet |

## FAQ

**Q: Apakah perlu login untuk mendengarkan radio?**
A: Tidak. Radio bisa didengarkan tanpa login.

**Q: Apakah perlu login untuk membaca artikel?**
A: Tidak. Artikel bisa dibaca tanpa login.

**Q: Bagaimana cara membuat akun?**
A: Buka halaman login > Klik Daftar > Isi formulir.

**Q: Apa keuntungan punya akun?**
A: Bisa menyimpan favorit, memberi komentar, dan fitur lainnya.

**Q: Bagaimana cara menghubungi radio?**
A: Buka menu Kontak atau lihat informasi kontak di website.

## Best Practice

1. **Bookmark website untuk akses cepat**
2. **Ikuti media sosial untuk info terbaru**
3. **Buat akun untuk fitur lebih banyak**
4. **Berikan feedback untuk perbaikan layanan**
5. **Gunakan koneksi internet yang stabil**
